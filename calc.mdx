## 1. 算术运算符

用于数学计算。

| 运算符 | 描述 | 示例（假设 `a=10, b=3`） |
| :-: | :-- | :-- |
| `+` | 加法 / 字符串拼接 | `a + b` → `13` |
| `-` | 减法 | `a - b` → `7` |
| `*` | 乘法 | `a * b` → `30` |
| `/` | 除法（整数除法截断小数） | `a / b` → `3` |
| `%` | 取余（仅支持整数） | `a % b` → `1` |

> You can also `a++` （单独一行）

## 2. 关系（比较）运算符

返回布尔值（`true` / `false`）。适用于可比较类型（整数、浮点、字符串、指针、通道、结构体等）。

| 运算符 | 描述 |
| :-: | :-- |
| `==` | 等于 |
| `!=` | 不等于 |
| `<` | 小于 |
| `<=` | 小于或等于 |
| `>` | 大于 |
| `>=` | 大于或等于 |

<Card title="需注意" type="note">
  比较 _小于或等于_ 或者 _大于或等于_ 中等号在后面；_等于_ 中是两个等号。
</Card>
